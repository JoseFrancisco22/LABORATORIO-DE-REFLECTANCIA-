# LABORATORIO-DE-REFLECTANCIA-
Código para controlar el sistema de medición de espectros de reflectancia diferencial (RDS) en el laboratorio de reflectancia.

## Introducción
Este repositorio contiene el código desarrollado en el laboratorio de reflectancia para adquirir y procesar espectros de Reflectancia Diferencial (RDS). El sistema se utiliza para caracterizar anisotropías ópticas en superficies y heteroestructuras semiconductoras, complementando las mediciones de fotoluminiscencia.

## Funcionalidades principales
- Control de monocromador
- Adquisición síncrona con voltimetro el lock-in amplifier
- Barrido automático de longitud de onda
- Cálculo en tiempo real de la señal RDS.
